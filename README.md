📌 Social – Blog Application

Social is a modern and responsive blog application where users can create, view, and manage posts. It focuses on simplicity, clean UI, and smooth user experience.

🚀 Features
✍️ Create and share posts
📄 View all available posts
🗑️ Delete posts
📱 Fully responsive design (mobile + desktop)
📂 Sidebar navigation with active state
🔔 Toast notifications for actions
🎯 Clean and user-friendly UI
🛠️ Tech Stack
Frontend: React.js
Styling: Bootstrap 5
State Management: React Context API
Icons: React Icons

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/Aj9630747827/Social.git
cd Social
2️⃣ Install dependencies
npm install
3️⃣ Run the app
npm run dev
